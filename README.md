# 基于egg-ts版本重写cnode.org小Demo

数据来源

> https://cnodejs.org/api

本地安装了redis，请求数据保存在redis中，不需要请求多次

旨在体验一下egg下的开发，ts使用还是不熟练

安装使用

> yarn install && yarn dev
